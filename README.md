# Obstacle-Detection-on-Railway-Track
電車の前方映像による線路上の障害物検出システム．

[処理例：動画](https://youtu.be/NS-7CSeqimQ)

## 使用ライブラリ
* OpenCV
* NumPy

```conda_requirements.txt```で環境構築すること．
